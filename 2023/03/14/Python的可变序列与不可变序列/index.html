Python的可变序列与不可变序列

發表於2023-03-14|更新於2023-03-14|Python
|閱讀量:

简介

在 Python 中，序列是一组按顺序排列的元素，包括字符串、列表和元组。Python 中的序列可以分为两种类型：可变序列和不可变序列。

可变序列指的是可以在原地进行修改的序列，例如列表（list）。这意味着你可以改变列表中的元素，添加新元素或删除元素。

不可变序列指的是不可以在原地进行修改的序列，例如字符串（str）和元组（tuple）。这意味着一旦创建了一个字符串或元组，就不能更改其中的元素。如果需要更改字符串或元组中的元素，则需要创建一个新的字符串或元组。

可变类型：

  • 列表（list）
  • 字典（dict）
  • 集合（set）
  • byte array（bytearray）
  • 用户自定义的类（在类中定义了可变对象）

不可变类型：

  • 数字（int、float、complex）
  • 字符串（str）
  • 元组（tuple）
  • frozenset（frozenset）
  • bytes
  • 用户自定义的类（在类中定义了不可变对象）

需要注意的是，即使某些类型是不可变的，但是如果它们包含可变类型的对象（例如，元组包含列表），则它们仍然可以被更改。此外，所有的对象都是通过引用进行传递的，因此，即使某些类型是不可变的，但是如果多个引用指向同一个对象，则更改任何一个引用所指向的对象的值都会影响其他引用。

扁平序列和容器序列

Python序列类型最常见的分类就是可变和不可变序列。但另外一种分类方式也很有用，那就是把它们分为扁平序列和容器序列。

扁平序列是指只能容纳一种数据类型的序列，例如字符串、字节串和元组等。扁平序列中的元素都是存储在一段连续的内存空间中，因此可以直接通过索引来访问。

容器序列是指可以容纳任意数据类型的序列，例如列表和集合等。容器序列中的元素可以是任意的 Python 对象，因此需要存储指向这些对象的引用。容器序列中的元素存储在不同的内存地址上，因此访问元素时需要进行一次间接引用。

按扁平序列和容器序列来分的话：

扁平序列：

  • 字符串：s = ‘Hello World’
  • 字节串：b = b’Hello World’
  • 元组：t = (1, 2, 3)

容器序列：

  • 列表：l = [1, ‘a’, True]
  • 集合：s = {1, 2, 3}
  • 字典：d = {‘a’: 1, ‘b’: 2, ‘c’: 3}
