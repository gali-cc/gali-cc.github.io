Python中的魔法属性和方法

發表於2023-03-13|更新於2023-03-14|Python
|閱讀量:

介绍

Python中的魔法方法指的是一类特殊的方法，它们使用双下划线（__）包围，用于实现对象的特定行为和功能。这些方法通常被称为“魔法方法”或“魔术方法”。通过实现这些特殊方法，自定义数据类型可以表现得跟内置类型一样，从而让我们写出更具表达力的代码——或者说，更具Python风格的代码。

以下是一些常见的魔法方法：

  • __init__(self, ...): 构造函数，用于创建对象并初始化它的属性。
  • __str__(self): 返回对象的字符串表示形式，通常用于打印对象或调试。
  • __repr__(self): 返回对象的字符串表示形式，通常用于开发者调试和交互式控制台中。
  • __len__(self): 返回对象的长度，通常用于集合类对象，如列表、元组、字典等。
  • __getitem__(self, key): 获取对象的元素，用于支持下标操作。
  • __setitem__(self, key, value): 设置对象的元素，用于支持下标操作。
  • __delitem__(self, key): 删除对象的元素，用于支持下标操作。
  • __getattr__(self, name): 获取对象的属性，用于支持动态属性访问。
  • __setattr__(self, name, value): 设置对象的属性，用于支持动态属性访问。
  • __call__(self, ...): 调用对象，用于将对象作为函数调用。

除此之外，Python中还有许多其他魔法方法，如比较运算符方法、数字运算方法、序列方法、迭代器方法、上下文管理器方法等。这些方法可以让开发者方便地实现自定义对象的行为和功能。

Python内建魔术属性和方法

以下是收集整理的魔法方法：

对象创建和初始化、销毁

  1. __init__(self, ...)：这是一个初始化方法，用于创建一个类的新实例，并初始化实例的属性。通常在这个方法中设置实例的属性，例如self.name = name。
  2. __new__(cls[, ...])：用于创建一个新对象，通常与__init__方法一起使用。
  3. __del__()：该方法在对象被销毁（即垃圾回收）时被调用。它可以用来执行对象的清理操作，例如关闭文件或释放资源等。
  4. __init_subclass__()：该方法是在定义一个类时调用的特殊方法。它可以用来为派生类（即继承自该类的子类）设置一些默认行为或属性。当一个类被定义时，Python会自动调用该类的__init_subclass__()方法，并将该类的子类作为参数传递给该方法。可以利用这个机制来对子类进行一些自动化设置，例如自动注册子类、自动添加属性等。

对象表示

  1. __str__(self)：这个方法返回一个字符串，用于描述对象。当你调用print(obj)时，它将打印出这个方法返回的字符串。
  2. __repr__(self)：这个方法也返回一个字符串，用于描述对象。通常用于调试和开发过程中。当你使用repr(obj)时，它将返回这个方法返回的字符串。

对象比较

  1. __eq__(self, other)：这个方法用于检查两个对象是否相等。通常用于判断对象是否相等，比如 obj1 == obj2。它返回True或False。
  2. __lt__(self, other)：这个方法用于比较对象的大小，用于小于号<操作符。它返回True或False。通常用于排序操作，比如使用sorted()函数。它返回True或False。
  3. __le__(self, other)：这个方法用于比较对象的大小，用于小于等于号<=操作符。它返回True或False。
  4. __ne__(self, other)：这个方法用于比较对象的不等性，用于不等号!=操作符。它返回True或False。
  5. __gt__(self, other)：这个方法用于比较对象的大小，用于大于号>操作符。它返回True或False。
  6. __ge__(self, other)：这个方法用于比较对象的大小，用于大于等于号>=操作符。它返回True或False。

数学运算

  1. __add__(self, other): 定义加法操作，当使用 “+” 运算符时自动调用。
  2. __sub__(self, other): 定义减法操作，当使用 “-“ 运算符时自动调用。
  3. __mul__(self, other): 定义乘法操作，当使用 “*” 运算符时自动调用。
  4. __truediv__(self, other): 定义除法操作，当使用 “/“ 运算符时自动调用。
  5. __floordiv__(self, other): 定义整除操作，当使用 “//“ 运算符时自动调用。
  6. __mod__(self, other): 定义取模操作，当使用 “%” 运算符时自动调用。
  7. __pow__(self, other[, modulo]): 定义幂运算操作，当使用 “**” 运算符时自动调用。
  8. __iadd__(self, other): 用于实现“就地加法”操作，也称为“增量赋值加法”。它与__add__方法的区别在于__add__返回一个新对象，而__iadd__会修改对象本身，因此被称为“就地”操作。具体地说，当我们使用“+=”运算符对一个对象进行加法操作时，如果该对象的类中定义了__iadd__方法，则Python会自动调用这个方法，实现对象自身的修改。
  9. __isub__(self, other): 实现“就地减法”操作，也就是“-=”运算符。
  10. __imul__(self, other): 实现“就地乘法”操作，也就是“*=”运算符。
  11. __itruediv__(self, other): 实现“就地除法”操作，也就是“/=”运算符。
  12. __ifloordiv__(self, other): 实现“就地整除”操作，也就是“//=”运算符。
  13. __imod__(self, other): 实现“就地取模”操作，也就是“%=”运算符。
  14. __ipow__(self, other[, modulo]): 实现“就地幂运算”操作，也就是“**=”运算符。

位运算符

  1. __and__(self, other)：用于按位与运算符（&）。
  2. __or__(self, other)：用于按位或运算符（|）。
  3. __xor__(self, other)：用于按位异或运算符（^）。
  4. __lshift__(self, other)：用于左移运算符（<<）。
  5. __rshift__(self, other)：用于右移运算符（>>）。

容器操作

  1. __len__(self)__：这个方法返回一个整数，表示对象的长度。通常用于获取对象的长度，比如len(obj)。注意，如果你的对象没有定义这个方法，len(obj)将会抛出TypeError异常。
  2. __getitem__(self, key)：这个方法使得对象可以像序列一样被索引和切片。当你使用obj[key]时，它将被调用。它通常用于实现自定义容器类。
  3. __setitem__(self, key, value)：这个方法用于设置对象的元素。当你使用obj[key] = value时，它将被调用。它通常用于实现自定义容器类。
  4. __delitem__(self, key)：这个方法用于删除对象的元素。当你使用del obj[key]时，它将被调用。它通常用于实现自定义容器类。
  5. __iter__(self)：这个方法使得对象可以被迭代。当你使用for循环迭代一个对象时，它将被调用。它通常返回一个迭代器。
  6. __next__(self)：这个方法用于迭代对象中的下一个元素。它通常在__iter__方法中返回一个迭代器时使用。
  7. __reversed__(self)：用于返回一个反转迭代器。
  8. __contains__(self, item)：这个方法用于检查一个对象是否包含某个元素。当你使用item in obj时，它将被调用。它返回True或False。

上下文管理

__enter__(self)和__exit__(self, exc_type, exc_val, exc_tb)：这两个方法用于实现上下文管理器。当你使用with语句时，__enter__方法将被调用，并返回一个上下文对象。当代码块执行完成时，__exit__方法将被调用，用于清理资源。

  1. __enter__(self)：用于进入上下文管理器。
  2. __exit__(self, exc_type, exc_value, traceback)：用于退出上下文管理器。

属性访问

  1. __getattr__(self, name)：这个方法用于获取对象的属性。当你访问一个不存在的属性时，它将会被调用。它返回一个值或抛出一个AttributeError异常。
  2. __setattr__(self, name, value)：这个方法用于设置对象的属性。当你设置一个属性时，它将会被调用。它通常用于对属性进行一些检查或者转换操作。
  3. __delattr__(self, name)：这个方法用于删除对象的属性。当你删除一个属性时，它将会被调用。它通常用于对属性进行一些检查或者转换操作。
  4. __getattribute__(self, name)：这个方法用于获取对象的属性。当你访问一个属性时，它将会被调用。它通常用于对属性进行一些高级操作，例如记录属性访问历史或者动态生成属性。

类型转换

  1. __bool__(self)：用于将对象转换为布尔值。当你使用if obj:或者bool(obj)时，它将被调用。它返回True或False。
  2. __int__(self)：用于将对象转换为整数。
  3. __float__(self)：用于将对象转换为浮点数。
  4. __complex__(self)：用于将对象转换为复数。
  5. __bytes__(self)：用于将对象转换为字节串。这个方法用于返回对象的字节表示形式。通常用于网络通信和文件操作。
  6. __str__(self)：用于将对象转换为字符串。

其他

  1. __call__(self, ...)：这个方法使得对象可以像函数一样被调用。当你调用一个对象时，它将会被调用。它通常用于实现一些比较复杂的逻辑操作。
  2. __hash__(self)：这个方法用于计算对象的哈希值。通常用于在使用集合或字典等需要哈希值的数据结构时。
  3. __dir__(self)：这个方法返回一个列表，包含对象的所有属性和方法。通常用于调试和开发过程中。
  4. __format__(self, format_spec)：这个方法用于格式化对象的字符串表示形式。当你使用str.format()时，它将被调用。

魔术属性

  1. __name__: 表示当前模块的名称。如果模块被直接执行，那么其值为__main__。
  2. __doc__: 表示当前对象的文档字符串，通常用于注释和文档说明。
  3. __module__: 表示当前对象所在的模块名称，通常用于模块间的导入和引用。
  4. __class__: 表示当前对象所属的类，通常用于对象的类型检查和动态绑定。
  5. __dict__: 表示当前对象的属性字典，包含对象所有的属性和方法，可以通过该字典进行动态操作。
  6. __bases__: 表示当前类的所有父类的元组，通常用于多重继承。
  7. __slots__: 表示当前类的属性名列表，可以用于限制对象的属性数量和属性名，提高内存使用效率。
  8. __file__: 表示当前模块的文件名，如果当前模块是内置模块或动态生成的模块，则其值为None。
  9. __closure__: 表示当前函数的闭包，包含函数内部的自由变量和其值。
  10. __annotations__: 表示当前函数或类的注解，包含函数参数、返回值和类成员的类型和说明。
  11. __qualname__: 表示当前函数或类的限定名称，包含所有嵌套作用域的名称。
  12. __globals__: 表示当前对象的全局命名空间，即定义该对象的模块的命名空间。
  13. __defaults__: 表示函数对象的默认参数值，以元组的形式返回，默认值为空元组。
  14. __code__: 表示当前函数对象的字节码对象，包含函数的指令和常量表。
  15. __builtins__: 表示当前模块的内置命名空间，即Python解释器的内置函数和常量。
  16. __kwdefaults__: 表示函数对象的关键字参数的默认值，以字典的形式返回，默认值为空字典。

魔法方法的数量是不固定的，它们是Python中的特殊方法，随着Python的版本升级和功能增强，魔法方法的数量也在不断增加。
