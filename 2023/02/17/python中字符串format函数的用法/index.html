基本用法

​ format() 是 Python 中一个非常强大和灵活的函数，用于将字符串中的占位符替换为指定的值，从而生成一个新的格式化后的字符串。format() 函数可以有多个参数，但第一个参数是必须的，它用于定义格式化字符串中占位符的位置和类型。后面的参数是可选的，用于替换占位符。下面是 format() 函数的语法：

在这个语法中，formatted_string 是要进行格式化的字符串，包含占位符 {0}、{1}、…、{n}。这些占位符可以被替换为对应的参数，即 arg1、arg2、…、argn。注意，占位符的位置是从 0 开始计数的。

​ 除了位置参数之外，format() 函数还支持其他几种类型的占位符，包括关键字占位符和格式化选项占位符。下面是 format() 函数的完整语法：
